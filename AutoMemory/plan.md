# AutoMemory

我们一起来实现一个长期记忆模块吧。我计划大致以类似"人类"的记忆策略来作为其算法:

- 提供所有短期记忆: 这些记忆刚刚发生
- 短期记忆的top-k类似记忆(Embedding->Query->Rerank): 这些记忆与当前可能有关

判断记忆的重要性与否集中于:

- 记忆经常被检索到: 这些记忆有可能是有用的
- 记忆被检索到后，输出的质量，比不被检索到输出的质量更好: 这意味着该记忆在某些情况下的输出是有效的
- AI自行拟定的记忆: 这些记忆通常是经验、总结等等
- 时间: 越古老的记忆越容易被遗忘

我在Github上找到了一些类似的仓库，他们也有类似的做法，例如:

- MemGPT提供MCP tool允许AI手动记忆，手动检索，自主调度
- PowerMem提供内置时间衰减机制
- Mem0支持时序推理

或许，我们能creative地将他们融合起来？创建一个能依据这样的策略提供记忆的harness:

- 提供tool允许AI自主调度
- 基于时间、重要性、逻辑链加权，提供能注入上下文的片段
- 提供接口允许批量添加记忆
- 能使用市面上的有效方法测试其记忆质量
